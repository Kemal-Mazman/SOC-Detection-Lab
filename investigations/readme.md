# Investigations

This folder contains investigation notes and analysis summaries from the SOC Detection Lab.

The purpose of this section is to document how suspicious activity was reviewed after detection logic fired in Splunk.

## Investigation Focus

Investigation notes may include:

* Alert summary
* Affected host
* Suspicious account or process
* Relevant Windows Security or Sysmon events
* Source IP or workstation information
* Timeline of observed activity
* Analyst interpretation
* Recommended response actions

## Current Investigation Themes

* Failed Windows logon activity
* Encoded PowerShell execution
* Persistence using scheduled tasks
* Registry Run Key persistence
* Windows discovery command sequences
* Local account creation and administrator group modification
* Local account admin + RDP attack-chain correlation

## Notes

Raw logs are not stored here by default. This folder is used for structured investigation write-ups, analyst notes, and summaries that explain what happened and how the activity was validated.
